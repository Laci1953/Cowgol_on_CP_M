The Cowgol include files and the library cowgol.coo

The cowgol.coo contains all the 1-byte, 2-byte and 4-byte arithmetic routines called by the cowgol implementation of its statements

The cowgolC.coh must be used, instead of cowgol.coh, when memory allocation calls are used.

Please consult also the LIBRARY folder for additional explanations about the use of the include files
